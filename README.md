## Add samples from images

Expanding samples from images.

### Structure

Create folders recursively like the following structure in the root path of this project.

```
├── images
│   ├── 0
│   ├── 1
│   ├── 2
│   ├── 3
│   └── 4
└── samples
    ├── 0
    ├── 1
    ├── 2
    ├── 3
    └── 4
```

### Classes

- 0 not phone
- 1 the feature phone
- 2 the smart phone
- 3 the back of phone
- 4 lighten of phone screen
